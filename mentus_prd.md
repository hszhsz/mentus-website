# 二、产品概述

## 2.1 产品背景

当前开源AI Agent（如OpenClaw）具备强大的本地执行能力，但存在上手门槛高、无统一账号体系、多端无法协同、功能偏通用、缺乏商业化闭环等痛点，难以满足普通用户“极简使用”和职场人“职业高效”的核心需求。Mentus基于OpenClaw内核进行产品化改造，弥补开源项目的产品短板，打造“普通人能用、职场人好用、能付费、能留存”的个人AI助理，实现“AI有脑（智慧决策、职业适配）、有手（落地执行、自动化操作）”的核心价值，契合垂直化AI Agent“深而精”的发展趋势，区别于通用AI助理“广而浅”的定位。

## 2.2 品牌含义

Mentus 源自“Ment（心智、大脑，对应AI的思考、决策、职业知识库能力）”与“Manus（手，对应AI的执行、操作、自动化能力）”的融合，精准传递产品“脑+手”的核心定位——不仅能像人类大脑一样思考、分析、决策，还能像人类双手一样落地执行、处理事务，既具备通用助理的便捷性，又拥有垂直职业助理的专业性。

## 2.3 核心价值

- 用户价值：极简上手（零技术门槛）、多端协同（一个账号打通全场景）、职业适配（专属Skill解决职场核心需求）、隐私可控（端云结合，数据自主）、持久安全（云端备份，避免数据丢失），解决“想得到但做不到、会操作但太繁琐、有需求但无适配”的痛点，同时弥补通用AI Agent在专业领域的知识盲区、场景错位问题。

- 商业价值：构建“免费基础使用+付费增值服务”的商业化闭环，依托职业Skill生态、会员体系实现盈利，打造差异化个人AI助理产品，抢占垂直化AI Agent市场份额，同时传承OpenClaw的开源基因，兼顾开源用户的使用习惯，扩大用户基数。

- 产品价值：将开源OpenClaw从“极客工具”转化为“大众产品”，打造“个人AI操作系统级助理”，形成“账号-端侧-云端-Skill生态”的完整产品体系，构建可持续迭代的产品壁垒。

# 三、核心目标（MVP阶段）

## 3.1 用户目标

- 降低上手门槛：新用户3分钟内完成“安装-登录-使用核心功能”全流程，无需配置API Key、无需修改代码，通过傻瓜式安装向导实现零成本上手。

- 覆盖核心场景：实现“日常琐事处理+基础职业需求”双覆盖，支持文件管理、日程管理、基础搜索等通用功能，以及1-2个核心职业（如律师、程序员）的基础Skill。

- 保障用户体验：多端协同响应延迟≤3秒，任务执行成功率≥90%，隐私保护符合行业标准，用户满意度≥85%。

## 3.2 商业目标

- 种子用户积累：MVP上线后1个月内，积累1000+有效注册用户，其中职业用户占比≥40%。

- 付费转化验证：付费Skill使用率≥20%，会员订阅转化率≥5%，完成商业化闭环验证。

- 生态初步搭建：接入10+基础免费Skill，3-5个付费职业Skill，完成Skill市场核心框架搭建，初步形成“基础+专业”的Skill分层体系。

## 3.3 产品目标

- 完成OpenClaw内核适配：实现内核与云端账号、多端端侧的无缝对接，保留其本地执行能力，优化其配置复杂度。

- 搭建核心产品体系：完成官网、多端端侧APP、云端管理后台、Skill市场的核心功能开发，实现各模块协同运行。

- 确立差异化优势：突出“职业垂直+端云协同+极简易用”三大核心差异，与通用AI助理、纯开源Agent形成明显区分，契合垂直化AI Agent的发展趋势。

# 四、用户画像

## 4.1 核心用户

25-45岁职场人（占比70%），涵盖律师、程序员、数据分析师、文案策划、行政人员等，具备明确的职业需求，时间成本高，追求高效办公，愿意为“能解决核心工作痛点的工具”付费，对AI工具接受度高，同时希望工具能贴合自身职业流程，弥补通用工具的专业不足。

- 核心诉求：职业相关任务自动化（如律师合同解析、程序员代码调试）、多端协同处理工作、高效管理工作事务、获取专业领域辅助支持，无需投入过多时间学习工具使用。

- 用户特征：具备一定的付费意愿（单月付费接受范围20-100元），注重隐私安全，对工具的专业性、稳定性要求高于便捷性，需要工具能精准匹配自身职业场景的工作流程。

## 4.2 次要用户

18-24岁学生、25-50岁普通职场人（占比30%），无专业技术背景，核心需求是处理日常琐事，对工具的易用性要求极高，付费意愿较低，仅在“能显著提升生活效率”的情况下考虑付费，对AI工具的专业度要求较低，更关注基础功能的便捷性。

- 核心诉求：日程提醒、文件整理、消息转发、基础搜索、语音助手等日常功能，无需复杂操作，能快速上手，支持多端同步，避免数据丢失。

- 用户特征：注重工具的免费性和易用性，对功能的深度要求不高，易被“极简操作”“多端协同”等亮点吸引，是免费用户的核心构成，也是潜在的付费转化群体。

## 4.3 潜在用户

中小企业团队、自由职业者，核心需求是“低成本实现任务自动化”“团队共享工具权限”，未来可通过企业版、定制化Skill拓展，其核心诉求是工具的可扩展性和团队协同能力，契合垂直化AI Agent“解决行业核心问题”的价值定位。

# 五、核心功能需求（按模块划分）

说明：功能按“云端模块+端侧模块+Skill生态模块”划分，优先实现MVP核心功能，迭代功能后续逐步拓展；所有功能均围绕“Mentus=脑+手”的核心定位，兼顾易用性、专业性和商业化，同时融入垂直化AI Agent的知识底座、场景适配特性。

## 5.1 云端模块（核心支撑，负责账号、付费、同步、Skill管理）

### 5.1.1 账号体系（必填功能）

- 功能描述：实现用户注册、登录、找回密码、账号设置等基础功能，一个账号打通所有端侧（电脑端、手机端），支持账号信息修改、退出登录、设备管理。

- 业务规则：
        

    - 注册方式：支持手机号、邮箱注册，无需强制第三方登录，注册后需完成实名认证（可选，用于付费、隐私保护）。

    - 登录验证：支持密码登录、验证码登录，可设置“自动登录”“设备绑定”，最多支持5台设备同时在线，支持远程退出其他设备登录。

    - 账号安全：密码采用加密存储，支持密码找回（手机号/邮箱验证），异常登录（异地、陌生设备）触发短信提醒，保障账号安全。

    - 隐私设置：用户可自主选择“云同步范围”（仅同步配置/同步配置+记忆/不同步任何数据），支持“账号注销”，注销后云端数据永久删除，端侧数据保留。

- MVP优先级：P0（核心必备，无账号体系无法实现多端协同、付费等核心功能）

### 5.1.2 付费管理（必填功能）

- 功能描述：实现用户充值、会员订阅、付费Skill购买、订单管理、发票管理等功能，支持多种支付方式，清晰展示付费权益。

- 业务规则：
       

    - 充值功能：支持微信、支付宝、银行卡充值，充值金额无门槛，充值后余额可用于购买会员、付费Skill，余额不支持提现，永久有效。

    - 会员体系：分为免费会员、基础会员（月付/年付）、高级会员（月付/年付），权益分层：
                

        - 免费会员：仅使用基础免费Skill、本地执行功能，不支持云端同步、多端协同，无模型额度赠送。

        - 基础会员：支持多端协同、云端同步、所有免费Skill，赠送基础模型额度（满足日常使用），可享受付费Skill折扣（8折）。

        - 高级会员：包含基础会员所有权益，赠送大额模型额度，免费使用部分付费Skill（每月可解锁2个），享受付费Skill折扣（6折）、优先技术支持、定制化Skill预约服务。

    - 付费Skill：支持“单次付费永久使用”“按调用次数付费”“按月订阅付费”三种模式，价格根据Skill的专业性、功能复杂度定价（10-200元不等），购买后绑定账号，所有端侧均可使用。

    - 订单管理：展示所有充值、会员订阅、Skill购买订单，包含订单号、金额、支付时间、状态（待支付/已支付/已取消/已退款），支持订单详情查看、发票申请。

    - 发票管理：支持申请电子发票（增值税普通发票），填写发票信息后，系统自动生成发票并发送至用户邮箱，支持发票历史查询、下载。

- MVP优先级：P0（核心商业化功能，支撑产品盈利闭环）

### 5.1.3 Skill市场（核心功能）

- 功能描述：实现Skill的展示、分类、搜索、安装、卸载、更新、评价、反馈等功能，分为免费Skill、付费Skill、定制化Skill三类，构建Skill生态，同时融入垂直化AI Agent的知识底座特性，提升Skill专业性。

- 业务规则：
        

    - Skill分类：按“日常通用”“职业垂直”“工具辅助”三大类划分，职业垂直类再细分律师、程序员、数据分析师等子分类，方便用户快速查找，每个职业Skill均配套专属领域知识底座，提升专业度。

    - Skill展示：每个Skill展示封面、名称、简介、功能详情、使用教程、价格（免费/付费）、用户评价、更新日志，支持“预览功能”（无需安装即可查看核心操作演示）。

    - 安装与卸载：用户在端侧可直接搜索、选择Skill，点击“安装”即可自动同步至云端账号，所有端侧同步生效；支持“一键卸载”，卸载后端侧、云端均移除该Skill，不影响其他功能，同时保留用户使用记录（可重新安装恢复）。

    - Skill更新：系统自动检测Skill更新，推送更新提醒，用户可选择“立即更新”或“稍后更新”，更新后保留用户之前的使用配置，同时优化Skill的专业度（同步更新领域知识底座）。

    - 评价与反馈：用户使用Skill后可进行评分（1-5星）、撰写评价，可反馈Skill的bug、功能建议，产品团队定期查看反馈，优化Skill，形成“用户反馈-产品优化- Skill迭代”的数据闭环。

    - 定制化Skill：支持用户提交定制化需求（如特定职业的专属功能、企业团队的定制化工具），产品团队评估需求后给出报价和开发周期，完成后推送至用户账号，仅该用户可使用，契合垂直化AI Agent“场景化定制”的特点。

    - Skill权限：所有Skill均采用“权限沙箱”机制，明确限定Skill可访问的资源（如仅读文档、仅操作浏览器、不可访问本地文件等），用户安装前可查看权限说明，保障用户隐私和设备安全，避免越权操作。

- MVP优先级：P0（核心差异化功能，支撑职业垂直定位，构建产品核心竞争力）

### 5.1.4 云端同步与持久化（必填功能）

- 功能描述：实现用户配置、Skill安装列表、任务历史、记忆数据（用户偏好、交互记录）的云端同步与备份，避免端侧文件丢失、设备更换导致的数据丢失，支持“一键恢复环境”。

- 业务规则：
       

    - 同步范围：仅同步“结构数据”（配置、列表、记忆、任务状态），不同步用户本地文件内容、屏幕内容、敏感隐私数据（如银行卡信息、身份证信息），保障用户隐私，呼应“AI在你本地跑，数据你自己管，云端只做同步”的产品定位。

    - 同步机制：默认实时同步（网络正常情况下），网络异常时，端侧数据暂存，网络恢复后自动同步；用户可手动触发“立即同步”“暂停同步”。

    - 持久化备份：云端数据自动备份（每日凌晨备份1次），备份数据保留30天，用户可手动触发“手动备份”，支持“恢复至指定时间点”的数据，避免数据误删。

    - 一键恢复：用户更换设备、重装端侧APP后，登录账号即可触发“一键恢复环境”，自动恢复所有Skill安装列表、用户配置、记忆数据，无需重新配置，快速恢复使用状态。

- MVP优先级：P0（核心体验功能，解决用户“数据丢失”的核心痛点）

### 5.1.5 云端管理后台（必填功能）

- 功能描述：供用户查看账号信息、付费权益、订单记录、Skill安装列表、同步状态，管理设备、隐私设置、模型额度，提交反馈与工单，是用户云端操作的核心入口。

- 业务规则：
        

    - 界面设计：简洁易用，分类清晰，核心功能（Skill管理、付费管理、同步管理）突出，支持搜索功能，方便用户快速找到所需操作。

    - 工单反馈：用户可提交功能建议、bug反馈、技术咨询，支持上传截图、描述详情，系统自动分配工单，产品/技术团队24小时内响应，反馈处理进度可实时查看。

    - 模型额度管理：展示用户当前模型额度（剩余额度、使用记录、到期时间），支持额度充值、额度赠送（会员权益），额度使用明细清晰可查（哪个Skill、哪个时间、使用多少额度）。

- MVP优先级：P0（核心支撑功能，衔接所有云端功能，提升用户管理效率）

## 5.2 端侧模块（核心执行端，负责功能落地、用户交互，分为电脑端、手机端）

### 5.2.1 通用功能（所有端侧均具备）

- 安装向导（必填功能）：
        

    - 功能描述：傻瓜式安装流程，≤3步完成安装与初始化，无需用户配置环境、填写API Key，降低上手门槛。

    - 业务规则：安装步骤为“安装客户端→登录账号→自动完成环境配置（适配本地系统、同步云端数据）”，每一步均有清晰指引，支持“跳过指引”（适合熟悉产品的用户），配置失败时自动提示错误原因并给出解决方案。

    - MVP优先级：P0（核心易用性功能，解决开源项目上手难的痛点）

- 账号登录与绑定（必填功能）：

    - 功能描述：支持账号密码、验证码登录，登录后自动绑定当前设备，同步云端数据（Skill、配置、记忆），支持“退出登录”“切换账号”。

    - 业务规则：登录后自动检测云端数据，同步至端侧；切换账号后，清空当前端侧的所有用户数据（保留客户端本身），登录新账号后同步新账号的云端数据；未登录账号时，仅支持本地基础功能（无云端同步、多端协同、付费Skill使用权限）。

    - MVP优先级：P0（核心支撑功能，衔接端侧与云端）

- 基础交互（必填功能）：
        

    - 功能描述：支持自然语言输入（文字、语音）、指令执行、结果反馈、任务管理，界面简洁易用，核心功能一键可达，适配不同端侧的操作习惯。

    - 业务规则：
                

        - 语音交互：支持中文、英文语音输入，语音转文字准确率≥95%，支持语音指令（如“帮我整理桌面文件”“提醒我下午3点开会”），适配手机端、电脑端的语音输入场景。

        - 指令执行：用户输入指令后，Mentus自动拆解任务、调用对应Skill、执行操作，实时展示执行进度（进度条、步骤提示），执行完成后反馈结果（文字、图片、文件等），执行失败时自动提示原因并尝试重试（最多3次），重试失败则给出手动解决方案。

        - 任务管理：展示当前执行任务、历史任务，支持“暂停任务”“终止任务”“重新执行任务”，历史任务可搜索、可查看详情（执行步骤、结果、使用额度），支持删除历史任务（删除后端侧、云端同步删除）。

    - MVP优先级：P0（核心交互功能，支撑用户日常使用）

- Skill管理（必填功能）：

# 六、交互流程（核心用户路径）

## 6.1 新用户注册-安装-使用核心流程（P0）

1. 用户访问Mentus官方站点，点击“注册”，选择手机号/邮箱注册，填写信息并完成验证，创建账号。

2. 注册完成后，用户在官网下载对应端侧（电脑端/手机端）APP，启动安装程序。

3. 按照安装向导完成安装（3步：安装→登录账号→自动配置），配置完成后进入端侧主界面。

4. 用户在端侧主界面，搜索并安装所需Skill（免费基础Skill/付费Skill，付费Skill需完成充值购买）。

5. 用户输入指令（文字/语音），Mentus拆解任务、调用对应Skill，执行完成后反馈结果，新用户流程结束。

## 6.2 多端协同流程（P0）

1. 用户在电脑端登录账号，安装所需Skill，执行任务（如文件整理、合同解析）。

2. 用户在手机端登录同一账号，自动同步电脑端的Skill安装列表、任务进度。

3. 用户在手机端下发指令（如“终止电脑端当前任务”“生成任务执行报告”），电脑端接收指令并执行。

4. 电脑端任务执行完成后，结果自动同步至手机端，用户在手机端查看结果、接收提醒，实现多端协同。

## 6.3 付费Skill购买-使用流程（P0）

1. 用户在端侧/Skill市场，搜索并查看目标付费Skill（查看功能、价格、评价）。

2. 用户点击“购买”，选择支付方式（余额/微信/支付宝/银行卡），完成支付。

3. 支付完成后，Skill自动安装至端侧，同步至云端账号，所有端侧均可使用。

4. 用户输入对应指令，Mentus调用该付费Skill，执行任务并反馈结果，用户可对Skill进行评价、反馈。

## 6.4 设备更换-数据恢复流程（P0）

1. 用户在新设备上，下载Mentus端侧APP，安装并登录原有账号。

2. 登录后，系统自动提示“是否恢复云端数据”，用户点击“确认恢复”。

3. 系统自动同步云端的Skill安装列表、用户配置、记忆数据，完成环境恢复。

4. 恢复完成后，用户可直接使用原有功能、Skill，无需重新配置，实现无缝切换。

## 7.1 易用性需求

- 安装流程：≤3步完成安装与初始化，无需用户配置环境、填写API Key，安装向导指引清晰，适配不同技术背景的用户。

- 反馈机制：操作反馈及时（≤1秒），任务执行进度清晰，错误提示明确（给出解决方案），避免用户迷茫，提升使用体验。

## 7.2 性能需求

- 响应速度：指令输入后，任务拆解响应≤1秒，简单任务（如日程提醒）执行完成≤3秒，复杂任务（如合同解析）执行完成≤30秒（根据任务复杂度调整）。

- 稳定性：端侧APP连续运行72小时无崩溃、无卡顿，任务执行成功率≥90%，云端服务可用性≥99.9%，无重大故障。

- 同步速度：云端与端侧数据同步≤3秒（网络正常情况下），网络异常时，恢复网络后10秒内完成同步，同步过程不影响用户正常操作。

- 资源占用：电脑端APP内存占用≤200MB，手机端APP内存占用≤100MB，不影响设备正常运行，避免过度占用CPU、内存资源。

## 7.3 兼容性需求

- 电脑端：支持Windows 10及以上、MacOS 12及以上、Linux（Ubuntu 20.04及以上）系统，适配不同屏幕尺寸、分辨率。

- 手机端：支持Android 10及以上、iOS 14及以上系统，适配不同品牌、不同屏幕尺寸的手机，支持横屏、竖屏切换。

## 7.4 隐私与安全需求

- 数据安全：用户密码采用MD5加密存储，云端数据采用AES-256加密传输与存储，防止数据泄露、篡改，符合国家隐私保护相关法规。
